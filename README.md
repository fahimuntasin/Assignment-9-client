# PawAdopt - Pet Adoption Platform (Client)

A full-stack pet adoption platform built with Next.js, MongoDB, Express, and Node.js.

## Live URL

[Deployed on Vercel](#)

## Features

- Browse all available pets with search by name and filter by species
- User authentication with email/password and Google login
- Submit adoption requests with pickup date and message
- Pet owners can add, update, and delete their pet listings
- Manage adoption requests with approve/reject functionality
- Responsive design with dark/light theme toggle
- Framer Motion animations for smooth UI transitions
- Secure JWT authentication with HTTPOnly cookies

## NPM Packages Used

- **next** - React framework
- **axios** - HTTP client
- **sonner** - Toast notifications
- **framer-motion** - Animations
- **next-themes** - Dark/light theme
- **lucide-react** - Icons
- **react-icons** - Social media icons
- **shadcn/ui** - UI component library
- **tailwindcss** - Utility-first CSS

## Getting Started

1. Clone the repository
2. Install dependencies: `npm install`
3. Create `.env.local` with `NEXT_PUBLIC_API_URL` pointing to your backend
4. Run development server: `npm run dev`
5. Open [http://localhost:3000](http://localhost:3000)
